üöÄ Render Deployment Steps

Step 1: Access Render Dashboard

  1. Visit https://render.com
  2. Sign in with your GitHub account
  3. Click the "Create +" button

Step 2: Deploy Backend Service

  1. Select "Web Service"
  2. Choose your library-project repository
  3. Configure the service:
    Name: library-backend Root Directory: backend Environment: Python 3 Build Command: pip install -r requirements.txt && python manage.py migrate && python manage.py collectstatic --noinput Start Command: gunicorn library_project.wsgi Plan: Free
  4. Add Environment Variables (see next step)

Step 3: Set Environment Variables (Backend)

Add these variables in the backend service settings:

SECRET_KEY= DEBUG=False PYTHONUNBUFFERED=1 ALLOWED_HOSTS=library-backend.onrender.com CORS_ALLOWED_ORIGINS=https://library-frontend.onrender.com

Generate SECRET_KEY:

python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"

Step 4: Create PostgreSQL Database

  1. Click "Create +" ‚Üí "PostgreSQL"
  2. Configure:
    Name: library-db Database Name: library User: library_user Region: (your preference) Plan: Free
  3. Copy the Internal Database URL

Step 5: Link Database to Backend

  1. Go back to your Backend Service settings
  2. Add Environment Variable:
    DATABASE_URL=
  3. Save changes (backend will redeploy)

Step 6: Deploy Frontend Service

  1. Click "Create +" ‚Üí "Static Site"
  2. Choose your library-project repository
  3. Configure:
    Name: library-frontend Root Directory: frontend Build Command: npm install && npm run build Publish Directory: build Plan: Free
  4. Add Environment Variable:
    REACT_APP_API_URL=https://library-backend.onrender.com
